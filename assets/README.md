# FigmaTokenImporter Brand Assets

## Logo Design

The FigmaTokenImporter logo represents the core concept of importing design tokens into Figma Variables through a minimalistic visual metaphor.

### Visual Elements

- **Dots (Left)**: Represent raw design tokens in various states - scattered and unorganized
- **Flow Arrow**: Shows the import/transformation process
- **Diamond (Right)**: Figma's iconic diamond shape, containing organized dots representing imported variables

### Color Palette

- **Primary Black**: `#0D0D0D` - Used for the Figma diamond
- **Gray Tones**: `#6B6B6B`, `#4A4A4A`, `#3A3A3A` - Used for dots and flow elements
- **Background**: `#FAFAFA` - Light gray circle background

### Available Formats

- `icon.svg` - Main vector logo (128x128px)
- `icon.png` - Standard resolution PNG (128x128px)
- `icon@2x.png` - Retina resolution PNG (256x256px)
- `favicon-24.svg` - Simplified favicon vector (24x24px)
- `favicon-24.png` - Favicon PNG (24x24px)

### Usage Guidelines

1. **Plugin Icon**: Use `icon.png` or `icon@2x.png` when publishing to Figma Community
2. **Web Favicon**: Use `favicon-24.png` for web interfaces
3. **Documentation**: Use `icon.svg` for scalable display in documentation
4. **Dark Mode**: Logo is optimized for dark backgrounds with light gray circular background

### Design Principles

- **Minimalistic**: Simple geometric shapes for clarity at small sizes
- **Meaningful**: Visual metaphor clearly represents token import functionality
- **Scalable**: Works from 24px favicon to larger display sizes
- **Consistent**: Aligns with Figma's design language and the plugin's UI design system
